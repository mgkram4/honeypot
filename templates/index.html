Security Information

Port Information

FTP (21)

File Transfer Protocol - Used for file transfers. Common target for brute force and anonymous access attempts.

SSH (22)

Secure Shell - Remote system access. Frequently targeted by credential stuffing and brute force attacks.

Telnet (23)

Legacy remote access protocol. Highly vulnerable due to lack of encryption.

HTTP (80)

Web traffic - Vulnerable to various attacks including SQL injection, XSS, and directory traversal.

HTTPS (443)

Encrypted web traffic - Still vulnerable to application-level attacks despite encryption.

MySQL (3306)

Database server - Target for SQL injection and unauthorized access attempts.

Attack Types

High Risk Attacks

Root access attempts, privilege escalation, system command execution

Medium Risk Attacks

Brute force attempts, reconnaissance, credential stuffing

Low Risk Attacks

Failed login attempts, basic port scans, non-malicious probes

Common Attack Patterns

SQL Injection

Attempts to manipulate database queries to gain unauthorized access or extract data.

Brute Force

Repeatedly trying different passwords to gain unauthorized access.

Directory Traversal

Attempting to access files outside the web root directory.

Command Injection

Inserting system commands to execute on the target server.

Credential Stuffing

Using stolen credentials to attempt unauthorized access.

Reconnaissance

Gathering information about the system for future attacks.
